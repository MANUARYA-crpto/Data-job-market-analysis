Data Jobs Market Analysis Dashboard

1. Project Overview
This project analyzes the global data job market to understand demand, salary trends, and geographic distribution of data-related roles such as Data Analyst, Data Scientist, and Data Engineer.

2. Tools & Technologies
- Power BI : Visualization and Dashboarding
- DAX : For measure like "Average year salary"
- Power Query : Data cleaning and transformation

3. Key Insights
- Total number of data-related job postings across regions
- Average yearly and hourly salaries
- Most in-demand data roles
- Geographic distribution of data jobs worldwide

4. Dashboard Preview : https://github.com/MANUARYA-crpto/Data-job-market-analysis/blob/main/Job%20dashboard.png

5. Business Use Case
- Helps job seekers understand market demand and salary benchmarks
- Assists recruiters and organizations in analyzing global hiring trends

6. Future Enhancements
- Add experience-level salary comparison
- Include time-based job trends
- Drill-through analysis by job role
